# Plotly Dashboard Assignment (240)

## Project overview

* Objective: Create a web app based dashboard that display visualizations that effectively describe the distribution of data from a dataset
* For raw project instructions see: [here](http://syllabus.africacode.net/projects/data-science-specific/dashboards/plotly_dashboards/)

## Prerequisites

- Python 3.x

## Installing repo

1. Clone the repository:
    ```bash
    git clone https://github.com/Thxbiso-DS/financial-service-uses-dashboard.git
    ```

2. Navigate to the directory:
    ```bash 
    cd financial-service-use-in-tanzania/
    ```

3. Create a virtual environment with a name of your choice or the given name:
   ```bash
   python -m venv plotly-env
   ```

4. Activate your environment:
   - On Windows:
    ```powershell 
    .\plotly-env\Scripts\activate
    ```
   - On macOS/Linux:
    ```bash
    source plotly-env/bin/activate
    ```

5. Install dependencies:
    ```bash 
    pip install -r requirements.txt
    ``````

## Viewing Dashboard

1. To view the dashboard, on your terminal simply type:
   ```bash
   python source/app.py
   ```

2. Click on the link provided. it should look something like this:
   ```
   http://127.0.0.1:8050/
   ```
